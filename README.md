# 合肥工业大学自动健康打卡脚本

本项目实现了最新的健康打卡系统的脚本，大家可以配合定时器使用。在Linux上可以使用systemd timer，在Windows上可以使用计划任务。

依赖安装：

```
pip install -r requirements.txt
```

查看帮助：

```
python hfuter.py -h
```
